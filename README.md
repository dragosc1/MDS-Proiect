# Proiect MSD - Belgradestra Dungeon(RPG Game)

Belgradestra Dungeon, un joc RPG developat de echipa formată din:

- Buca Mihnea-Vicentiu (Grupa 252)
- Dragoș Ciobanu (Grupa 251)
- Ricardo-Dumitru Petrovici (Grupa 251)
